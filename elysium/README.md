# Elysium — Monospaced Pterodactyl Theme

A terminal-native dark theme for Pterodactyl Panel, inspired by [OpenCode](https://opencode.ai)'s design language — 100% monospaced, flat-with-hairline, ASCII-bracket aesthetic.

## Quick Install

```bash
curl -sL https://raw.githubusercontent.com/instax-dutta/elysium/main/install.sh -o /tmp/elysium.sh && sudo bash /tmp/elysium.sh
```

## Features

- 100% JetBrains Mono typography — every label, button, table cell, and badge
- Near-black canvas (`#121212`) with warm-white ink (`#e8e6e3`)
- Flat design — no drop shadows, only 1px hairline borders
- 4px radius on every interactive element
- ASCII-bracket markers (`[+]` / `[-]`) as UI glyphs
- Apple HIG semantic color ramp for status feedback
- Dark themed React client + AdminLTE admin panel

## After Install

Clear browser cache or use incognito mode to see the theme.

## Uninstall

Restore from backup at `/var/www/elysium-backup-YYYYMMDD-HHMMSS/`
